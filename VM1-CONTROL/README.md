# VM1-CONTROL

> Those playbook are only meant to be used with **Ubuntu** based distros. \
> You need to have **ansible** installed in order to use those playbooks.

## Run

To run a specific playbook , you can use the command below :

```bash

ansible-playbook <playbook_file_name>.yml 

```
Those playbooks are used to configure the **host** and not other VMs.

## Playbooks

- *playbook-terraform* : add terraform repo and install terraform on the VM
